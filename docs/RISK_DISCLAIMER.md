## 📋 核心风险揭示

### 1. 市场风险

#### 1.1 股市波动风险
- A股市场波动较大，策略可能在极端行情下产生重大亏损
- 黑天鹅事件（如金融危机、政策突变、战争等）可能导致策略完全失效
- 市场流动性枯竭时，可能无法按预期价格买卖

#### 1.2 系统性风险
- 整个市场系统性下跌时，分散投资也难以避免亏损
- 宏观经济政策变化（如利率调整、汇率波动）可能对策略产生不利影响
- 交易所技术故障或临时停市可能导致无法及时平仓

#### 1.3 个股风险
- 个股停牌、退市、ST 等情况可能导致资金长期被锁定或损失
- 财务造假、重大违规等"黑天鹅"事件可能导致个股暴跌
- 涨跌停板限制可能导致无法按预期价格买卖

### 2. 技术风险

#### 2.1 策略失效风险
- **过拟合风险**：历史回测表现优异的策略在未来可能完全失效
- **市场环境变化**：市场结构、参与者行为变化可能导致策略不再适用
- **参数敏感性**：策略参数微调可能导致结果巨大差异

#### 2.2 系统故障风险
- **软件 Bug**：本项目为开源项目，可能存在未发现的代码缺陷
- **数据错误**：第三方数据源（AkShare/Tushare）可能存在数据缺失或错误
- **网络故障**：网络延迟、中断可能导致交易指令无法及时执行
- **硬件故障**：服务器宕机、断电等可能导致系统无法运行

#### 2.3 LLM 不确定性风险
- **幻觉问题**：大语言模型（LLM）可能生成看似合理但实际错误的策略代码
- **理解偏差**：LLM 可能误解用户意图，生成不符合预期的交易逻辑
- **知识局限**：LLM 的金融知识可能过时或不准确
- **性能不稳定**：LLM 在不同市场环境下表现差异巨大

### 3. 操作风险

#### 3.1 配置错误风险
- 参数配置错误（如仓位比例、止损阈值）可能导致意外损失
- API Key 配置错误可能导致数据获取失败或交易失败
- 数据库配置错误可能导致数据丢失或系统崩溃

#### 3.2 过度优化风险
- 对历史数据过度优化可能导致策略对未来市场不适应
- 频繁调整策略参数可能导致"追涨杀跌"效应
- 盲目追求高收益可能忽视风险控制

#### 3.3 心理风险
- 过度依赖系统可能导致忽视人工判断的重要性
- 连续亏损可能导致情绪化操作或放弃系统
- 连续盈利可能导致过度自信和加大杠杆

### 4. 合规风险

#### 4.1 监管政策风险
- 程序化交易监管政策可能趋严，导致部分策略无法继续使用
- 申报频率限制可能导致高频策略无法执行
- 穿透式监管可能要求披露更多策略细节

#### 4.2 交易规则风险
- 券商可能对程序化交易账户实施更严格的限制（如提高保证金）
- 交易所可能对异常交易行为实施临时停牌或限制交易
- 跨市场交易可能涉及更多合规要求

#### 4.3 税务风险
- 高频交易可能产生更高的交易成本和税费
- 不同交易品种（股票、ETF、期权）的税收政策不同
- 用户需自行申报交易所得并缴纳税款
